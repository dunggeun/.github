## 반가워요! 👋

`둥근`은 단 한 명으로 구성된 개발팀이에요.

둥근 팀은 **스스로 학습하고 성장하는 것** 이라는 한 가지 원칙만을 따르고 있고,  
이를 실천하며 여러 사용자에게 가치를 제공하는 것을 목표로 하고 있어요.

_이 가치가 비록 사소하고 작은 가치일지라도 말이죠._
